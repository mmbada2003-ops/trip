🚄 城市間交通
上海 → 蘇州
建議搭乘高鐵。
出發站：上海虹橋站
抵達站：蘇州站
車程：約 25－35 分鐘
建議提早 30－45 分鐘抵達車站。
🗺️ 上海虹橋站 🗺️ 蘇州站
蘇州 → 杭州
建議搭乘高鐵。
出發站：蘇州站
抵達站：杭州東站
車程：約 1.5 小時
熱門時段建議提前購票。
🗺️ 蘇州站 🗺️ 杭州東站
杭州 → 上海
建議搭乘高鐵。
出發站：杭州東站
抵達站：上海虹橋站
車程：約 45－70 分鐘
抵達上海後可轉乘地鐵前往靜安區。
🗺️ 杭州東站 🗺️ 上海虹橋站
✈️ 機場交通
上海浦東國際機場 → 市區
方案一：地鐵 2 號線
價格較便宜，但車程較長。

方案二：磁浮列車＋地鐵
可先搭磁浮至龍陽路，再轉乘地鐵。

方案三：滴滴／計程車
適合多人同行或行李較多。
🧭 導航至南京路
上海市區 → 浦東國際機場
回程航班時間為 20:10。
建議最晚 16:00 左右離開新天地。
國際航班建議提前至少 3 小時抵達機場。
若遇尖峰時段，請預留更多交通時間。
🧭 導航至機場
🚇 市區交通建議
上海：以地鐵為主，景點間距離較遠。

蘇州：可搭地鐵、計程車，平江路與園林區適合步行。

杭州：西湖周邊建議步行或搭公車，跨區可使用地鐵與計程車。

建議使用支付寶或微信支付搭乘大眾運輸。
📌 搭車提醒
1. 高鐵進站通常需要安檢與證件查驗。

2. 請準備台胞證或可使用的旅行證件。

3. 熱門班次建議提前購票。

4. 上下班尖峰時段，地鐵可能十分擁擠。

5. Google Maps 在中國部分地區資訊可能不完整，可搭配高德地圖或百度地圖使用。
